평소 BOJ 문제를 구름 IDE로 풀 때 작성한 파일을 루트 디렉토리에 문제번호만 적고 저장한다.

이렇게 되면 작성한 파일 제목으로만 봐서는 내가 푼 문제인지, 어떤 알고리즘 분류의 문제인지 알 수 없게 되는 문제가 발생한다.

이를 계기로 루트 디렉토리에 있는 모든 .py 파일들을 알맞은 디렉토리에 이동시키는 프로그램을 구상함

simulate)
1. 루트 디렉토리에 있는 파일을 순서대로 접근
2. 파일의 제목에 있는 문제 번호를 가지고 웹 크롤링을 이용하여 해당 문제의 분류를 구한다.
3. 만약 디렉토리 중 해당 분류가 존재하지 않을 시 디렉토리를 만들어서 이동시킨다.
4. 이동시킨 후 루트 디렉토리에 남은 파일을 삭제한다.

주의해야 하는 부분)
1-1. .py 확장자만을 취급한다.
2-1. 11049_nr 같은 파일이 있기 때문에 문자열 slice를 주의해야함
2-2. 분류가 1개가 아닌 문제가 많기 때문에 모든 분류 디렉토리당 하나씩 파일을 복붙한다.
